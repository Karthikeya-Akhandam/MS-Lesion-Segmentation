## Note on Citation Format

The above references use IEEE style:
- Journal papers: Author(s), "Title," *Journal Name*, year.
- Conference papers: Author(s), "Title," in *Conference Name*, year, pp. X–Y.
- For the final paper, add volume/issue/DOI numbers where available from the source PDFs.
